⚠️ Eclectyc Energy - Setup Incomplete

If you're seeing this page, the application is not properly configured.

The Eclectyc Energy platform requires initial setup before it can run. You're seeing this static HTML page instead of the application because:

  • PHP dependencies haven't been installed (missing vendor/ directory)
  • Files may be in the wrong location (deployment path issue)
  • PHP may not be properly configured in Apache

Common Issues & Solutions

Issue 1: Missing Dependencies

Symptoms: "Failed to open vendor/autoload.php" error

Solution: Run composer install

cd /path/to/eclectyc-energy
composer install --no-dev --optimize-autoloader

OR use the helper script:

bash deployment/install-dependencies.sh

Issue 2: Wrong Upload Location

Symptoms: Error logs show public/public/index.php paths

Solution: Files were uploaded to wrong directory

See DEPLOYMENT_PATH_ISSUE.md for detailed fix instructions

Issue 3: PHP Not Processing

Symptoms: Seeing this HTML instead of dynamic content

Solution: Ensure PHP is configured in Apache and index.php is in DirectoryIndex

The public/.htaccess file should contain:

DirectoryIndex index.php

Setup Steps

  1. Install Dependencies: 
    cd /path/to/eclectyc-energy
bash deployment/install-dependencies.sh
  2. Verify Structure: 
    php deployment/fix-deployment-structure.php
  3. Configure Environment:

    Ensure .env file exists with correct database credentials

  4. Set DocumentRoot:

    In your hosting panel (Plesk/cPanel), set DocumentRoot to:

    /path/to/eclectyc-energy/public
  5. Test:

    Reload this page - you should see the login screen instead of this message
